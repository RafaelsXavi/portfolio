
# Portfólio | Rafael S. Xavier

> Portfólio profissional desenvolvido em Next.js, React, TypeScript, Tailwind CSS e Framer Motion. Interface moderna, minimalista e responsiva, focada em performance, clareza e experiência premium para recrutadores e tech leads.

## ✨ Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Deploy: Vercel

## 🚀 Como rodar localmente

```bash
# Instale as dependências
pnpm install

# Rode o servidor de desenvolvimento
pnpm dev

# Acesse http://localhost:3000
```

## 📁 Estrutura

- `app/` — Páginas (Home, Projetos, Sobre, Contato)
- `components/` — Componentes reutilizáveis (Header, Footer, ProjectCard)
- `public/` — Imagens e assets

## 💡 Diferenciais

- Layout escuro, elegante e responsivo
- Animações leves e hierarquia visual clara
- Código limpo, organizado e pronto para evolução
- Foco em UX/UI para mercado tech
